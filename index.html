Welcome to HexEmbed

HexEmbed is a simple tool designed to help you easily embed color swatches into any IFrame-embed capable tool. Whether you're organizing notes, building a visual reference, or designing your workspace, HexEmbed allows you to define colors, view their names and hex codes, and quickly copy them in various formats.

Try it

URL Guide:

Simply define your colors and how you'd like them displayed by using the URL query parameters.

Example URL:

?colors=110068,fe0024,ff4901,f5e400,49fe00&style=listv
  • colors – Provide one or more hex color codes, separated by commas.
    • Don't include the # (hash symbol); just the 6-digit code.
    • Example: ff0000 for red, ffaa55 for light orange, 00ff00 for green.
  • style – Choose how you want to view your colors.
    • swatches – displays a grid of color boxes.
    • listv – displays a vertical list of colors.
    • listh – displays a horizontal list of colors.

You can customize your color experience by adjusting these parameters directly in the URL.

Available Shortcuts:

While interacting with the color swatches, you can copy and export the information you need with the following shortcuts when clicking a color item:

  • Left Click – Copy Hex Color to Clipboard: A simple click on any color will copy its hex code to your clipboard.
    For example: #ff0000
  • Ctrl / ⌘ + Left Click – Copy CSS Variable and Color: Will copy both a CSS variable (based on the color name) and the hex code to your clipboard.
    For example: --color-red: #ff00000;
  • Shift / ⇧ + Left Click – Copy Color Name: Will copy just the name of the color to your clipboard.
    For example: Red
  • Alt / ⌥ + Left Click – Copy CSS Variable Name Only: Will copy just the color’s name in the format of a CSS variable.
    For example: --color-red
